1. plan.mdProject OverviewAetherLink is a distributed system designed to grant Alexa voice-control over a local Windows machine via direct IPv6 communication. It bypasses third-party tunnels (like Ngrok) to ensure low latency and data privacy.System ArchitectureVoice Entry: Alexa Skill (hosted on Amazon Developer Console).Relay Logic: AWS Lambda function acting as a secure HTTP client.Transport Layer: Direct IPv6 (TCP) on a custom high-range port.The Brain (Local): Python-based FastAPI server running as a background service.Media Engine: yt-dlp for stream extraction and MPV or VLC for headless playback.Security ImplementationProtocol: IPv6 with a strictly defined port.Authentication: X-Aether-Key (UUID4) header-based verification.Sandboxing: Commands are restricted to a predefined list of directories and scripts to prevent arbitrary code execution (ACE).2. phase.mdPhase 1: Connectivity & Network (The Foundation)IPv6 Verification: Confirm a global-routable address (Prefix 2xxx:).Router Config: Open a specific port (e.g., 58008) in the router's IPv6 firewall pointing to the PC's MAC/Interface.External Test: Verify connectivity via a Python listener script.Phase 2: Local Agent Development (The Engine)API Framework: Scaffold FastAPI with routes: /status, /execute, /play.OS Integration: Map voice keywords to system paths (e.g., "Andromeda" → C:\Users\User\Dev\Andromeda).Shell Interface: Implement a controlled subprocess.run wrapper for CMD/PowerShell execution.Phase 3: YouTube & Media LogicStreamer: Implement a logic that takes a search query, uses yt-dlp to find the best audio stream, and pipes it to a headless player.Audio Control: Interface with the Windows Core Audio APIs to adjust system volume via voice.Phase 4: Alexa Interaction ModelSkill Building: Define Intents (OpenDirectory, RunCommand, PlayMusic).Slot Filling: Train Alexa to recognize specific project names and command variables.Lambda Link: Write the Lambda function to translate Alexa JSON into AetherLink API calls.3. roadmap.mdMilestoneObjectiveKey Deliverable01: The HandshakeEstablish secure IPv6 commsSuccessful "Ping" from external IPv6 to
